Flask 基础 

python环境的安装

pipenv 集成了pip + virtualenv + requirement.txt ，使得生成独立的开发环境更加便捷

具体操作

	pip install pipenv 

	pipenv install flask

这样生成独立的开发环境

* 只要使用pipenv安装，就能将安装包放到虚拟环境中；

* 只有需要在全局环境下安装、更新和删除包，才使用pip；
